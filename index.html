This is spacefically the design and layout. It is essentially the box that wraps around every html element.
It consists of margins,boarders,padding,and the actural content

PROPERTIES OF CSS BOX MODEL

Padding

Clears an area around the content. The padding is transparent.

Widith

It specifies the width of the element's content area.

Margin

Clears an area outside the border. The margin is transparent.

Border

Clears an area outside the border. The margin is transparent.

Content

The content of the box, where text and images appear.
